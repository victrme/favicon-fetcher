
# favicon-fetcher

Favicon fetcher finds sites favicons and fetches them fast

### Endpoint

```HTTP
GET /:url
```

### How to use

#### Netlify Edge functions
```bash
# install
npm install --global netlify-cli

# debug
netlify dev
```

#### Cloudflare Workers
```bash
# install
npm install --global wrangler

# debug
wrangler dev

# deploy
wrangler login
wrangler deploy
```
